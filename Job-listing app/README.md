# Finance Job Listings

A simple job listing web app built with React and TypeScript.

## What It Does

- Displays a list of 10 finance-related jobs
- Each job shows the title, company, location and salary
- Click **"Show Details"** to see more about a job
- Click **"Hide Details"** to collapse it back

## Built With

- React
- TypeScript
- Vite
- CSS

## How To Run It

**Step 1** — Clone or download the project

**Step 2** — Install dependencies
```
npm install
```

**Step 3** — Start the app
```
npm run dev
```

**Step 4** — Open your browser and go to
```
http://localhost:5173
```

## Project Structure

```
src/
├── App.tsx        # Main component, all job data and logic lives here
├── App.css        # Styles for the job cards
├── index.css      # Global styles for the whole page
└── main.tsx       # Entry point, don't touch this
```
